# 🕰️ TimeTravel Agency — Webapp Interactive

Projet réalisé dans le cadre du module *IA Générative & Développement Web*.  
Cette webapp présente une agence de voyage temporel fictive et propose une interface immersive permettant de découvrir différentes époques, interagir avec un chatbot IA et (optionnellement) réserver un voyage.

---

## 🚀 Fonctionnalités

### 🎨 Interface & Navigation
- Landing page immersive avec hero animé
- Navigation fluide entre les pages (Home, Destinations, Chat, Booking)
- 3 destinations temporelles :
  - Paris 1889
  - Ère du Crétacé
  - Florence 1504

### 🧠 Agent Conversationnel
- Chatbot IA intégré
- Conseils personnalisés selon les préférences de l'utilisateur
- Mini FAQ automatisée

### 📆 (Optionnel) Réservation
- Formulaire simple : destination, dates, voyageurs
- Validation basique

### 🌀 Animations
- Fade-in des sections (Framer Motion ou AOS)
- Hover effects sur les cards
- Micro-interactions sur les boutons

---

## 🛠️ Stack Technique

- **Framework** : React / Next.js
- **UI** : Tailwind CSS
- **Animations** : Framer Motion (ou AOS)
- **Routing** : Next.js Router
- **Chatbot mock** : logique frontend simple (pas d’API obligatoire)
- **Build & Dev** : Bolt.new ou Cursor
- **Déploiement** : Vercel
